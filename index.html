Task Manager

Welcome to the Task Manager project! This project includes a Python program that allows administrators and users to manage tasks effectively.

About the Program

The task_manager.py script allows users to:

  • Register new users (admin only)
  • Add tasks
  • View all tasks
  • View tasks assigned to themselves
  • Display statistics (admin only)
